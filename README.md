# 💄 Lira Makes

Projeto de e-commerce desenvolvido como trabalho em grupo no curso <a href="https://github.com/dvinacioo/PROZ-Curso-T-Des.-Sistemas-">Técnico em Desenvolvimento de Sistemas</a>.

## 🚀 Sobre o projeto

O **Lira Makes** é um site de venda de produtos de maquiagem, com foco em acessibilidade, design moderno e experiência do usuário.

O projeto simula uma loja virtual com funcionalidades básicas de compra.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## ⚙️ Funcionalidades

- 🛒 Carrinho de compras funcional
- 💰 Cálculo automático do total
- 🎠 Carrossel de produtos com Bootstrap 5
- 📌 Header fixo ao rolar a página
- 📱 Links de WhatsApp e Instagram ilustrativos (pendentes de configuração)  
- 🔗 Links do rodapé ilustrativos< (não funcionais)
